Fijo de altura completa Side Nav

Trate de desplazarse esta área , y ver cómo el sidenav se pega a la página

Notice that this div element has a left margin of 25%. This is because the side navigation is set to 25% width. If you remove the margin, the sidenav will overlay/sit on top of this div.

También notamos que nos hemos fijado desbordamiento : auto para sidenav . Esto agregará una barra de desplazamiento cuando el sidenav es demasiado largo ( por ejemplo, si tiene más de 50 enlaces dentro de ella ) .
